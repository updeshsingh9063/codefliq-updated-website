# Codefliq - Custom Web & App Development Studio

Codefliq is a premium, high-performance website designed to showcase web development, custom app building, and AI integration services. Built with a modern aesthetic, rich micro-animations, responsive design, and immersive media.

## Tech Stack
- **Core**: HTML5, Vanilla JavaScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide Icons
- **Fonts**: Clash Display (Premium Display Font), Geist (Default/Sans-Serif), Great Vibes (Cursive)
- **Media**: Immersive video landing page and dark mode background effects

## Features
- **Hero Header**: Full-screen looping video header with premium typography.
- **Interactive Video Section**: "What We Do" section showcasing developers at work with custom controls.
- **Interactive Cards**: Staggered scroll-reveal entrance and hover-active glow/scale animations on all testimonial and process cards.
- **Smooth Scroll Navigation**: Seamless in-page navigation links.
- **Responsive Layout**: Optimized for mobile, tablet, and desktop screens.
- **Contact Form**: Beautiful glassmorphic contact card with quick connect links.

## How to Run Locally
1. Clone this repository:
   ```bash
   git clone https://github.com/updeshsingh9063/codefliq-updated-website.git
   ```
2. Navigate to the project directory:
   ```bash
   cd codefliq-updated-website
   ```
3. Open `frontend/index.html` in any modern web browser or run it via a local development server (e.g., Live Server in VS Code, or Python's `http.server`).
